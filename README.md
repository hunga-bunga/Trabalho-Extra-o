# Extração de Conhecimento de Dados Biológicos – Trabalho Prático 1

Este repositório apresenta o trabalho realizado na unidade curricular **"Extração de Conhecimento de Dados Biológicos"**, do Mestrado em **Bioinformática** da **Universidade do Minho** (ano letivo 2025/2026). Inclui os dados e relatórios do Trabalho Prático 1, centrando-se na análise de dados de **expressão genética**.  

O projeto consiste no desenvolvimento de um **workflow** em **R/Bioconductor**, abrangendo desde o **processamento inicial dos dados** até à **análise exploratória** e **modelação preditiva**.

## Conjunto de Dados

- **Fonte:** [cBioPortal](https://www.cbioportal.org/study/summary?id=prad_tcga)  
- **Tipos de Dados:** Expressão Génica (RNA-seq)  
- **Dimensão:** 500 amostras
