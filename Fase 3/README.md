# Extração de Conhecimento de Dados Biológicos – Trabalho Prático 1 - Terceira fase do trabalho

## Descrição

Nesta pasta encontra-se a versão final do Trabalho Prático 1, correspondente à **terceira fase**, integrando todos os desenvolvimentos realizados ao longo do projeto. Assim, o trabalho encontra-se **completo** nesta versão.

## Conteúdo Adicionado

Nesta fase, foram incorporadas abordagens de **análise preditiva** e **aprendizagem automática**, incluindo:

* **Classificação de amostras biológicas**, com aplicação de diferentes modelos de machine learning;
* **Modelos de regressão** para análise de relações entre variáveis;
* **Comparação do desempenho de pelo menos dois modelos**, recorrendo a métricas adequadas;
* **Seleção de características (features)** com foco em dados biológicos;
* **Análise da importância de genes**, com o objetivo de identificar os mais relevantes para o problema em estudo.

## Objetivo

O principal objetivo desta fase foi enriquecer a análise exploratória anterior com técnicas preditivas, permitindo não só compreender os dados, mas também **modelar e prever padrões biológicos relevantes**.

## Notas Finais

Esta versão representa a integração final de todas as etapas do trabalho, incluindo preparação de dados, análise exploratória e modelação preditiva.
